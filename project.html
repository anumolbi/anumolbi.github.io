Course Overview - Topics in Computer Systems (001)

COurse Instructor: Phu Phung

Course Summary

In this course, students will study basic web application development with front-end (HTML5, JavaScript, CSS) and back-end (PHP/MySQL). Secure programming principles and practices will be introduced to avoid potential web application vulnerabilities and attacks.

Course Activities

- Lab 1: Introduction to HTML5 and CSS
- Lab 2: JavaScript Essentials
- Hackathon: Secure Web Application Development

Overview

This course provides an in-depth understanding of web application development and security. Students will learn about client-side and server-side technologies, version control, and secure programming principles. The course emphasizes hands-on exercises, labs, and a hackathon to apply theoretical knowledge in practical scenarios.

Projects

Suicide Ideation Using Deep Learning

Developed a deep learning model to detect suicidal ideation in social media posts, achieving 95% accuracy. Extracted and cleaned a large dataset from social platforms to train classification algorithms, prioritizing users needing mental health support. Published research paper on suicide ideation methodology and ethical considerations.

Telecom Churn Prediction

Developed a predictive model to identify customers at high risk of churning using Python and sklearn, achieving 92% accuracy in predicting churn 1 month before cancellation using gradient boosting. Identified the most influential factors driving churn through exploratory data analysis and presented recommendations to leverage predictions to proactively offer retention incentives.

School Tuition Fee Management System

Developed a web application to manage school tuition fees and payments, including an online portal for parents to view fees, download invoices, and make payments. Integrated payment gateway APIs to enable online fee submission via cards/netbanking and generated custom reports and analytics on collection rates, defaulters, and revenue. Helped improve fee collection rates and reduce delays for partner schools.
